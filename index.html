HTML E CSS

Dia 1, data: 17/10/2024

Hoje eu aprendi sobre flex-shrink e flex-grow que são usadas no elemento filho do conteiner. O flex-shrink define se um elemento pode ou não encolher de acordo com o tamanho da tela, seu padrão é "flex-shrink: 1;", isso significa que ele pode encolher, para que ele não encolha basta colocar o número 0. Caso voce coloque outro número positivo além de 1, o elemento irá encolher ainda mais. O flex-grow define quanto o elemento pode crescer, seu padrão é "flex-grow: 0;", neste caso significa que o elemento não pode crescer. Para que o elemento cresça basta por o número 1. se você quiser que o elemento ainda mais, é só por outro número positivo maior que 1...

JavaScript

Dia 1, data: 18/10/2024

Meus estudos em JavaScript já estão relativamente avançados, tenho uma boa base em logica de Programação e isso tem facilitado tudo. Eu já aprendi as estruturas condicionais (if e else), e no momento estou iniciando nas estruturas de repetição. Hoje eu aprendi as estruturas while e do while. A estrutura while é basicamente repetir um determinado conjunto de instruções enquanto uma condição for verdadeira. As duas estrturas são muito parecidas, a diferença é que a while fica no inicio, enquanto do while se localiza no final da operação.
